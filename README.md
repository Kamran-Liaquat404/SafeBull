# GhostFirewall

**GhostFirewall** is a modern cybersecurity and password management application designed to help users generate strong passwords, analyze password strength, and stay informed about security best practices. Built for ethical hacking enthusiasts, programmers, and security-conscious users.

---

## Features

- **Password Generator**: Create short, unique, and professional passwords.
- **Password Analyzer**: Check the strength of your existing passwords.
- **Security Tips**: Learn essential cybersecurity practices.
- **Dark / Light Theme**: Switch between themes with persistent settings.
- **Responsive Design**: Works on both desktop and mobile devices.
- **Future Enhancements**: Plans to add advanced security features like 2FA guidance and phishing awareness alerts.
